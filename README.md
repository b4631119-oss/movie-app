# 🎬 Movie App (Разбор и рефакторинг)

Это учебный проект на React (Vite, Redux Toolkit, Tailwind CSS), который я скачал для детального разбора логики стейт-менеджмента и асинхронных запросов .

## 🛠 Что внутри:
* **Движок:** React + Vite
* **Логика данных:** Глобальный стейт на Redux Toolkit, запросы через Axios к API TMDB
* **Стили и переходы:** Tailwind CSS + кастомный сплэш-скрин на лоадерах



## 🚀 Как запустить у себя:

1. Установи зависимости:
   ```bash
   npm install


2. Запусти проект
   ```bash
   npm run dev
